DocuMon DRMS

Document Request Monitoring System — Mapua Malayan Colleges Mindanao, Registrar's Office

A full-stack web system that replaces the Registrar's Office's fragmented Excel-based monitoring with centralized request tracking, token-based multi-office clearance coordination, and real-time student status updates. Built across four development phases and validated through User Acceptance Testing with the Assistant Registrar and Registrar Director.

Next.js 14 TypeScript Django 4.2 PostgreSQL Supabase Vercel Render
Live System Frontend Repo Manuscript (PDF)
📋
13-Stage Workflow

Official document request workflow confirmed with the Registrar's Office — from Pending through Claimed or Shredded.

🔗
Token-Based Clearance

Shareable clearance links for up to 11 offices — no login required. RO staff controls link validity and regeneration.

🆔
Document Request No.

Auto-generated IDs in format RO-YYYY-MM-DOCCODE-NNN with monthly sequence reset.

⚙️
Automated Background Jobs

Daily cron jobs via cron-job.org handle overdue payment flags and 90-day shredded document triggers.
